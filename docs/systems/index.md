| [Caching](caching.md) | Cache strategies, eviction policies, CDNs, distributed caching |
| [Databases](databases.md) | SQL vs NoSQL, ACID, indexing, sharding, replication |
| [Load Balancers](load-balancers.md) | L4 vs L7, routing algorithms, health checks |
| [Distributed Systems](distributed-systems.md) | CAP theorem, consistency models, consensus, partitioning |
