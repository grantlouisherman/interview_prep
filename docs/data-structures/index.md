# Data Structures

Fundamental data structures every engineer should know cold. Each page covers the structure's definition, key operations with time/space complexity, implementation patterns, major algorithms, use cases, and practice problems.

| Structure | Key Property | Common Interview Use |
|-----------|-------------|---------------------|
| [Stacks](stacks.md) | LIFO ordering | Parentheses matching, DFS, undo/redo |
| [Queues](queues.md) | FIFO ordering | BFS, task scheduling, buffering |
| [Heaps](heaps.md) | Min/max access in O(1) | Top-K problems, median finding, priority scheduling |
| [Binary Trees](binary-trees.md) | Hierarchical structure | BST search, traversals, balanced trees |
| [Graphs](graphs.md) | Arbitrary relationships | Shortest path, connectivity, topological sort |
